# Сервис "Оставить отзыв"

Небольшой PHP-сервис для сбора отзывов клиентов.

Форма позволяет:

* поставить оценку от **1 до 5**
* оставить **необязательный комментарий**

Отзыв сохраняется в базу данных вместе с `client_id`.

---

# Требования

Для запуска проекта необходимо установить:

* **Docker**
* **Docker Compose**
* **Make**

---

# Запуск проекта

### 1. Собрать контейнеры

```bash
make build
```

### 2. Запустить проект

```bash
make up
```

### 3. Установить зависимости Composer

```bash
make vendor
```

После этого приложение будет доступно в браузере.

---

# Остановка проекта

```bash
make down
```

---

# Просмотр логов

```bash
make logs
```

---

# Проверка функционала

После запуска проекта откройте в браузере:

```
http://localhost/?client_id=1
```

Доступные тестовые `client_id`:

```
http://localhost/?client_id=1
http://localhost/?client_id=2
http://localhost/?client_id=3
http://localhost/?client_id=4
http://localhost/?client_id=5
```

Ожидаемое поведение:

* отображается форма оценки
* можно выбрать оценку от **1 до 5**
* можно оставить комментарий
* после отправки отображается сообщение об успешной отправке

---

# Проверка несуществующего клиента

Откройте:

```
http://localhost/?client_id=999
```

Ожидаемый результат:

```
Ссылка на голосование недоступна, свяжитесь с нами по телефону
```
